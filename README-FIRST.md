# Comp Compass — quick upload (5 minutes)

Hi Efrat! Please add these 4 files to your repo. All from the browser, no setup.

## Steps

1. Unzip this file. You'll see 4 files:
   Dockerfile, .dockerignore, cloudbuild.yaml, next.config.ts
   (ignore this README — don't upload it).

2. Go to: https://github.com/EfratIzaks/Comp-Compass

3. Make sure the branch says **main** (top-left dropdown).

4. Click **Add file** → **Upload files**.

5. Drag all 4 files into the page.
   (next.config.ts will replace the old one — that's correct.)

6. In the box at the bottom, type a message like: Add Cloud Run deploy

7. Click **Commit changes**.

Done! That's all you need to do. Amit takes care of the rest on Google Cloud.

(One more thing may pop up later: GitHub might ask you to "approve" or
"install" the Google Cloud Build app on the repo — just click approve when
Amit lets you know.)
